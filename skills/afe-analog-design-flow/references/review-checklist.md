# Review Routing (Deprecated Combined Checklist)

The former combined checklist mixed public-skill publication review with AFE
project gate review. Those activities now have separate authority and records:

- Use `publication-review.md` when installing, packaging, or publishing this
  skill. It has no authority over a design project gate.
- Use `project-gate-review.md` for an independent AFE project review. The
  machine-readable gate review record must conform to
  `governance/schemas/gate-review.schema.json`.

This routing file is not a checklist, evidence record, independent review, or
gate authorization. `candidate_report_check.py` cannot satisfy either review.
